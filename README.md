TYPING PRACTICE APLLICATION
   -->A simple typing speed assessment application built using HTML, CSS, and JavaScript.
   Users can practice typing random words and measure their WPM and accuracy using HTML,CSS,
   JavaScript.
   
 Overview
     It is a web-based application that helps users improve their typing skills by measuring typing speed and accuracy in real time. The application presents users with text passages to type, tracks their input against the original text, and calculates performance metrics such as **Words Per Minute (WPM)**, **accuracy**, and **errors**. Built using **HTML, CSS, and JavaScript**, the project demonstrates concepts such as DOM manipulation, event handling, timers, and real-time data processing through an interactive and user-friendly interface.

Objectives
   -Develop an interactive platform for practicing typing skills.
   -Measure typing speed using the Words Per Minute (WPM) metric.
   -Calculate typing accuracy by comparing user input with the reference text.
   -Provide instant feedback on typing performance, including errors and completion time.
   -Improve user engagement through a responsive and intuitive interface.
   -Apply core web development concepts to build a real-time application.

Features:
  ->WPM calculation
  ->Acuracy Percentage
  ->Restart option
  ->Correct Word validation
  ->Random Word generation
